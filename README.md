# Supply Chain Performance Analysis using SQL

## Project Overview

This project analyzes supply chain performance using SQL to evaluate supplier efficiency, delivery performance, product quality, and logistics costs. 

The objective is to identify operational trends and performance differences between suppliers across different regions. The analysis helps businesses understand supplier reliability, transportation efficiency, and cost optimization opportunities.

This project simulates a real-world supply chain scenario similar to automotive manufacturing supply chains.

---

## Tools Used

SQL  
Excel (for dataset creation)

---

## Dataset Description

The dataset contains supply chain shipment records with the following fields:

supplier_id – unique supplier identifier  
supplier_name – supplier company name  
supplier_region – geographic location of supplier  
product_category – type of component supplied  
delivery_time_days – number of days required for delivery  
defect_rate – percentage of defective components  
shipment_quantity – number of units shipped  
transport_cost – logistics cost for shipment  
shipment_date – date of shipment

---

## Business Questions

This project answers the following operational questions:

1. Which supplier regions ship the highest volume of products?
2. Which suppliers have the fastest delivery times?
3. Which product categories have the highest defect rates?
4. What are the transportation costs by supplier region?
5. Which suppliers maintain the lowest defect rates?
6. What are the monthly shipment trends?

---

## Key Insights

German suppliers demonstrated the lowest defect rates and fastest delivery times.

Chinese suppliers handled the largest shipment volumes but showed higher defect rates.

Indian suppliers maintained balanced delivery performance and moderate transport costs.

Transport cost per shipped unit varied significantly across suppliers, highlighting opportunities for logistics optimization.
